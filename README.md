# EAppx/EMsix unpacker/decryptor

NOTE: Work-in-progress project!

## Build

Requirements:

- Rustup / Cargo

```
cargo build
```

## Usage

NOTE: Currently only commands `unpack` / `unbundle`/`info` are implemented

Check usage with

```
makeappx unpack --help
makeappx unbundle --help
makeappx info --help
```
